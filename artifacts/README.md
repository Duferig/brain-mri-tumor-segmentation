# Artifacts

Эта папка хранит:

- `manifests/` — train/val/test manifests для BraTS.
- `models/` — лучшие веса baseline и improved моделей.
- `metrics/` — history CSV, итоговые JSON, comparison CSV/Markdown.
- `predictions/` — результаты инференса и preview PNG.
- `uploads/` — временные пользовательские загрузки для API.
- `runs/` — дополнительные экспериментальные логи.

Сами бинарные веса не добавляются в репозиторий.
