# Структура пояснительной записки

## Глава 1. Анализ предметной области

- Клиническая значимость выявления и сегментации опухолей головного мозга.
- Особенности МРТ и мультимодальных данных.
- Обзор U-Net, attention-подходов и transformer-подобных архитектур.
- Краткое описание `BraTS`.

## Глава 2. Постановка задачи и требования

- Цель и критерии качества.
- Входные данные и ожидаемые выходы.
- Архитектура `обучение -> инференс API -> веб-демо`.
- Ограничения исследовательского прототипа.

## Глава 3. Реализация

- Формирование manifests и валидация данных.
- Предобработка и аугментации.
- Baseline `3D U-Net`.
- Improved `SwinUNETR`.
- FastAPI сервис и Streamlit-интерфейс.

## Глава 4. Эксперименты

- Дизайн train/val/test split.
- Dice по `WT`, `TC`, `ET`.
- `HD95`.
- Сравнение baseline и improved моделей.
- Анализ ошибок, limit cases и направления улучшения.
